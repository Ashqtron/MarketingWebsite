Join a Bank Queue from anywhere...

Save time by limiting the amout of time you spend queing up for a service at a bank Branch

The App Features

Here are the awesome things that our App can do

  • Register

    New users can register to the App

  • Login

    Returning users can login quickly

  • Home

    From the Home screen, Users can acces several Options

  • Rating system

    Userc can rate Branches to aid other users on satisfactory customer service

  • Visit a Branch

    Choose a closeby Bank to visit from the Map

  • Select a Branch Service

    Select a Service from the listed active services

  • View and Join Queue

    View real time bank Queue and request to Join Bank Queue

  • Get Queue Token and Go to Branch

    Save your Queue Token and travel to Branch within specified time
